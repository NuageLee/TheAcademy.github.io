Hexo+GitHub+MWeb搭建个人博客（Mac平台）

博客地址：www.theacademy.top

Hexo 安装

Hexo官网：https://hexo.io/zh-cn/
官网上有详细的安装指南

这时候就在blog文件夹安装好了hexo框架，开启hexo server后，在http://localhost:4000/.端口就可以看到自己的本地博客了。
然后，使用VSCode打开博客文件夹来看一看：

scaffolds 脚手架，工具模板
scripts 写文件的js，用来扩展hexo功能
source 存放博客文章
source/_drafts草稿箱
source/_posts 博客文章
themes 存放主题配置文件
_config.yml 全局配置文件
db.json 静态常量

主题选择

框架搭建好了，当然是选一个自己喜欢的主题，官网里有很多主题，并且都配备了安装和配置指南
我选择的是这款主题：

这款主题的配置特殊一些，直接在github上下载下一个主题文件，然后再在主题文件夹里安装一次hexo即可。之后便可以在http://localhost:4000/端口看到自己更换好主题以后的博客了，主题的信息设置一般会在主题的github里有详细说明。

上传到GitHub

此时的博客智能在本地4000端口查看，可以将其上传到GitHub服务器上，GitHub为每一个仓库设计了一个接口，可以添加对该项目的介绍网页，我们利用这一功能，将博客上传为介绍网页，就可以通过GitHub地址进行访问了。

新建仓库


仓库的名字一定要为：your blog’s name.github.io
之前看了很多文章说可以不用.github.io，亲测如果不加加载不出网页样式，所以还是一定要加上.

修改_config.yml文件

用vscode打开博客文件夹，修改_config.yml文件下的deploy配置：

生成静态页面

关联好git以后，在终端输入：
hexo g
生成静态页面

执行后博客文件夹内会生成一个public文件夹，就是生成的静态文件了，把它上传到github即可：
hexo d
上传到github，第一次上传需要你在终端内输入git的名字和密码
之后也可以使用命令：
hexo g -d
直接生成上传

修改GitHub Pages

回到github中你的博客的仓库，进入仓库的Settings,在GitHub Pages 那一栏的Source选择对应的分支并保存

这是绑定好域名后的，绑定域名之前只需要选择sourse里的branch就可以了，
-master branch 会把/README.md作为网站的index.html
-master branch/docs folder 会把/docs/README.md作为网站的index.html
这里选择的是master branch然后save
到这一步你就已经可以进入”username.github.io/yourblogname“查看你的博客了，如果不想绑定域名的话使用vscode打开博客文件夹，修改source下的CNAME文件，为上面的地址，就可以使用了，还有一个小问题，这时候你打开你的博客网站是没有样式的，要把_config.yml里的root改为/yourblogname.github.io。之后就可以使用了。

绑定域名

我是在阿里云上购买的域名，.top的域名很便宜，第一年九块钱，看到很多小伙伴有优惠码三块钱买成的。我直接买了三年的，大概60多。这里注意一下由于17年公布的法律现在买域名都需要实名认证，所以注册阿里云之后先去实名认证了再来买域名吧。
购买后可以在域名控制台看到购买的域名

点击右边的解析
一开始我只添加了A解析，但是每次上传到git都会收到一条邮件，提示我将README绑定为我的github地址。
在stackoverflow上查看说要创建四个A解析和一个CNAME解析，但是阿里云A解析会和CNAME解析发生冲突。所以在这里只创建了CNAME解析，问题还是解决了。
新建解析

创建后保存，接下来打开vscode来修改_config.yml的URL

将source下CNAME里写上自己的域名，修改后上传博客就可以了
注意，每次上传博客前要先清楚之前生成的public文件，每次更新博客的时候执行下面命令就行：
hexo clean
hexo g -d
之后进入github下仓库的Settings下的GitHub Pages，会看到：

这时就可以通过你的域名访问你的博客了。

使用MWeb写博客

MWeb是一款Mac下的MarkDown编辑器，hexo静态博客的图片上传是个大问题，很多博主表示本地上传虽然可行，但是太太太麻烦了，所以这里推荐使用MWeb配合七牛云来写博客
七牛云的注册也不多说了，但是也是需要实名认证。。。真的麻烦得很啊
七牛云据说有5G的空间，对个人用户来说足够了，这也是这么多云平台大家都用七牛云的原因吧。
注册之后点击右上角的 管理控制台

然后选择左边栏的对象存储
新建存储空间，输入一个空间名字即可
打开MWeb -> 打开偏好设置 -> 发布服务，选择七牛云
空间名称：七牛云创建的空间名称
AK和SK：在七牛云的个人面板里有个秘钥管理，里面有对应的AK和SK
域名：username.github.io/yourblog.github.io其实就是上面你通过github地址访问博客的地址。
在MWeb中，command+E打开外部模式，这里会弹出一个编辑窗口，在右边栏中左下角点加号，引入文件夹，添加博客文件夹下的source文件夹：

到这一步，大功告成，我们便可以在_posts文件夹下新建Markdown来编写自己的文章了。
编写完后还是那几部：
1.cd到博客目录下
2.hexo clean
3.hexo g -d
你编写的博客便可以上床到github，并通过绑定的域名进行访问啦。
注意，不要忘了使用MWeb编写的时候要在前面添加文章设置，具体可以看你主题里自带的文章头部是什么样子的，把他们复制过来修改一哈
