结构化数据的分类

机构化数据可分为数值型数据和分类数据两类，对于数据科学家来说一定要将数据先做分类，为下面的数据清理工作打下基础。

连续型数值数据

如风速、持续时间等

离散型数值数据

某一事件的发生次数

分类数据

分类数据主要可分为二元数据和有序数据，二元数据：例如取值为0或1，true或false等；有序数据：例如数值排序1、2、3、4、5。

矩形数据

矩形数据就是所谓的表结构数据，表中每一行为一个样本（记录），每一列为一种特征（变量）。

Python中的矩形数据

R中的矩形数据为data.frame对象

一些非矩形数据结构

数据大部分都是矩形数据结构，但是还是存在一些非矩形结构，主要有时间数据、空间数据、图形三种。

时间数据

时间数据记录了对同一目标的连续测量值。在物联网数据中站主要比例。

空间数据

坐标数据，比矩形结构更加复杂

图形

中位数

使用中位数进行位置确定的原因是对于一些数据来说均值对离群值更敏感，比如一个区域的收入估计，如果这个区域中住着比尔盖茨或者巴菲特，则中值则更稳健，切尾均值也可以达到同样的效果。

变异性估计

位置只是总结特性的一个维度，另一个维度是变异性(variability),也称离差(dispersion)。它测量了数据集是紧密聚集的还是离散的。
变异性是统计学的一个核心概念，统计学关注：

  • 如何测量变异性
  • 如何降低变异性
  • 如何识别变异性中的随机性
  • 如何识别真是变异性的各种来源
  • 如何在存在变异性的情况下作出决策

基于偏差的变异性估计

偏差也叫误差或残差，首先要选择使用基于均值的偏差还是基于中值的偏差
栗子：给定依据数据集[1,4,4]，则

  • 均值=3
  • 中值=4
    则
  • 均值偏差为[1-3,4-3,4-3]
  • 中值偏差为[1-4,4-4,4-4]

这些偏差说明了数据围绕均值/中值的分散程度

平均绝对偏差

变异性的测量就是估计这些偏差的典型值。在数据科学中就是预测值和真实值的距离
单纯的相加是没有意义的，因为和总为零。因此引入了平均绝对偏差

方差

方差和标准差是最广为人知的变异性估计量，他们都是基于偏差的平方。
方差，也叫做均方误差是偏差平方的均值

###标准差
标准差是方差的平方根

标准差在机器学习中用的更普遍，因为从数学角度看，使用平方值要比使用绝对值更方便。
###中位数绝对偏差
中位数绝对偏差(MAD)是一种稳健(对于离群点)的变异性估计量，公式为：

有时中位数绝对偏差会乘上一个常数比例因子，通常为1.4826，使得在正态分布下，中位数绝对偏差与标准差具有相同的尺度。

上述四种变异性估计的总结

即使数据符合正态分布，上述四种变异性估计也不尽相同，标准差总是大于平均绝对偏差，平均绝对偏差总是大于中位数偏差

基于百分位数的变异性估计

另一种估计方式是查看有序数据的分布，称为极差，即数据最大值与最小值之间的差。
极差对离群点很敏感。为了避免这一点，我们可以先删除两端的10%。
寻找百分位数首先要对数据进行排序，然后找到各个阶段的百分位数，一种常用的方法是寻找四分位距。

四分位距=75百分位数-25百分位数

饼状图

众数

指数据中出现最频繁的一个或一组数值

期望值

期望值是一种加权均值，权重使用的是类别出现的概率
计算步骤如下：

  • 输出值 x 输出值出现的概率
  • 将这些值加起来
    举个例子，一个公司提供一种包月服务，一种是每月300美元，5%客户会选择；另一种50美元，15%会选择，另外80%不会选择任何服务。
    则，参加会议的人支付服务费的期望值为：
    期望值=0.05x300 + 0.15x50 + 0.8x0 = 22.5

相关性

  • 正相关：变量X的高值随变量Y的高值变化而变化
  • 负相关：变量X的高值随变量Y的低值变化而变化

相关系数（皮尔逊相关系数）

相关系数用于给出了两个特征之间相关性的估计值，最常用的是皮尔逊相关系数：

其他统计学的相关系数有，斯皮尔曼秩相关系数、肯德尔秩相关系数，但是在数据科学中不常用。
可绘制相关性图标来查看各个变量之间的相关性，在R语言中使用corrplt包来绘制
